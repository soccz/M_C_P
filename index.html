class WorkSession:
    def __init__(self, goal):
        self.goal = goal      # plan.md 목표
        self.done = []        # handoff.md 완료
        self.blocked = []     # handoff.md 막힌 점

    def plan(self):           # plan.md 작성
        print(f"목표: {self.goal}")

    def verify(self, checks): # verify.md 검증
        for c in checks:
            print("✅" if c["ok"] else "❌", c["name"])

    def handoff(self):        # handoff.md 작성
        print(f"완료: {self.done}")
        print(f"막힘: {self.blocked}")

3개 저장점

각 저장점에서 중간 확인 → 다음 단계

1 기반 설정
CLAUDE.md (리뷰 규칙) + settings.json (읽기 전용) + Skill (code-review)
확인: /memory에서 규칙 로드, /review 수동 실행
2 MCP + SubAgent
MCP로 git 정보 연결 + SubAgent A(보안) + SubAgent B(스타일) 병렬 실행
확인: /review → 병렬 리뷰 보고서 생성
3 Scheduled Task + 알림
Cloud Scheduled Task 정기 실행 + Slack 커넥터로 결과 알림
확인: 자동 리뷰 실행 + Slack에 요약 도착
핵심 설계: 리뷰 시스템은 src/ 수정 불가 (settings.json deny) → 안전한 자동화
